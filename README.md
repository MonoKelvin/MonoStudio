## 功能特性

### 个人生活
- **灵感碎片**：捕捉转瞬即逝的想法，快速记录灵感
- **工作手记**：记录工作中的点滴，支持富文本编辑
- **随想录**：随性的思考与感悟，支持标签分类

### 实用工具
- **密码生成器**：生成安全的随机密码，支持自定义长度和字符类型
- **单位转换**：长度、重量、温度、面积、体积等多种单位转换
- **时间工具**：时间戳转换、时间差计算、倒计时功能
- **文件哈希**：计算文件的MD5、SHA-1、SHA-256、SHA-512哈希值

### 系统工具
- **桌面图标管理**：查看、重命名与组织桌面图标
- **系统文件搜索**：按关键字快速定位文件
- **系统信息**：查看系统硬件和软件信息（CPU、内存、磁盘、网络）
- **网络工具**：Ping测试、IP地址检测、DNS查询、网速测试、User-Agent解析
- **进程管理**：查看和管理系统进程，支持结束进程

### 开发工具
- **GUID生成器**：生成UUID v4格式的唯一标识符，支持批量复制
- **JSON格式化**：格式化和验证JSON数据，支持语法高亮
- **Base64工具**：Base64编码和解码
- **正则表达式测试**：测试和验证正则表达式，支持实时匹配

### 媒体工具
- **图片转换**：图片格式转换与质量压缩，支持JPG、PNG、WebP格式
- **视频转换**：视频格式转换和处理
- **音频工具**：音频格式转换和编辑
- **屏幕截图**：支持全屏截图、窗口截图和区域截图，带有暗色背景和十字线辅助，支持矩形选择和自由绘制模式，截图后可保存或复制到剪贴板

### 学习工具
- **日语学习**：五十音图学习与词汇记忆（N5-N1等级分类）

### 系统设置
- **应用设置**：主题切换、外观调整、快捷键设置
- **个性化设置**：颜色方案、字体大小调整
- **关于信息**：版本信息、隐私政策、赞助支持

## 安装和运行

1. 克隆仓库：
   ```bash
   git clone https://github.com/MonoKelvin/MonoStudio.git
   cd MonoStudio
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 开发模式运行：
   ```bash
   npm run dev
   ```

4. 构建生产版本：
   ```bash
   npm run build
   ```

## 项目结构

```
/
  ├── electron/          # Electron主进程代码
  │   ├── main.js        # 入口文件
  │   ├── preload.js     # 安全上下文
  │   └── utils.js       # 工具函数
  ├── src/               # 前端源代码
  │   ├── components/    # 通用组件
  │   │   └── base/      # 基础组件（按钮、输入框、对话框等）
  │   ├── modules/       # 功能模块
  │   │   ├── shell/     # 应用外壳（标题栏、侧边栏、工作区）
  │   │   └── tools/     # 工具模块（按功能分类）
  │   ├── common/        # 公共服务
  │   │   └── services/  # 服务层（存储、Toast、对话框）
  │   ├── config/        # 配置文件
  │   ├── assets/        # 静态资源（图标、图片、数据）
  │   └── styles/        # 全局样式（主题变量、基础样式）
  ├── dist/              # 构建输出目录
  └── package.json       # 项目配置
```
